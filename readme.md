Hello and Welcome to my ToDo App!
========================================================================================

- This application will allow you to add & view tasks that you input into the terminal.

- You also have the ability to delete tasks after they have been inputted.

- Quitting the ToDo app is as simple as typing 'quit'.

- Error handling gives you a smooth and seamless experience.

========================================================================================
How to get started:
========================================================================================

- Open the todo.py file.

- Run the file using the 'Run Python File' button on the top right of the window.

- Follow the instructions given by the application in the terminal below!
